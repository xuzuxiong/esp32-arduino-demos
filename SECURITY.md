# 安全策略 / Security Policy

## 支持的版本

| 版本   | 支持状态 |
|--------|----------|
| 0.1.x  | 早期维护 |

## 报告漏洞

如果你发现与本仓库相关的安全问题（例如硬编码凭据、远程代码执行风险），请**不要**公开 Issue。

请通过 GitHub Security Advisories 或仓库 Owner 的私有渠道联系维护者，并说明：

- 问题描述与影响范围
- 复现步骤（如适用）
- 建议修复思路（可选）

## 维护者承诺

- 确认后会在合理时间内回复。
- 修复后会发布补丁版本并在 CHANGELOG 中记录（不含 exploit 细节）。

## 使用者须知

- 本固件示例**不包含**生产级安全加固。
- MQTT、OTA、WiFi 功能需部署在你自己可控的网络环境中。
- 切勿在公开仓库或 Issue 中粘贴真实密码、Token 或内网地址。
